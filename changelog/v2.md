# v2.0.0

架构重构 — 统一 profiles 架构，新增 parse 命令。

## 破坏性变更

- **统一 profiles 架构**：移除原先的 envs/configs 分离结构，合并为统一的 `profiles` 节。原先的 `envs` 和 `configs` 不再使用。

## 新增

- `cc parse` 命令：将现有 settings JSON 文件解析为 profile 配置，支持 `-c` 复制到剪贴板
- 统一 profile 格式，每个 profile 可包含 `env`、`permissions`、`hooks` 等所有 Claude Code 设置字段

## 修复

- 更新所有测试用例以匹配统一架构
