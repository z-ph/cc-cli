# v2.0.3

新增 serve 反代、CLI 参数透传、交互优化。

## 新增

- `cc serve` 命令：本地 HTTP 反向代理服务，支持模型路由替换
  - 子命令：`start`、`list`、`stop`、`stop-all`、`log`
  - `--run` 模式：启动代理后自动拉起 Claude Code
  - 异步日志队列，支持批量写入和 flush
  - Worker 进程 IPC 通信和 SIGTERM 优雅关闭
- `cc <id>` 支持透传 Claude CLI 参数（如 `cc myprofile -c` → `claude -c --settings <path>`）
- `cc add` 环境变量选择器升级为 autocomplete 搜索 + 全量枚举（~200 个 Claude Code 环境变量）
- `cc add` 选择器改为左右分页，Custom/完成常驻顶部
- `cc edit` 交互流程与 add 同步，删除 env 改为 checkbox 选择

## 修复

- 移除 launch 命令中的 base 合并，避免全局配置优先级过高
- `cc parse` / `cc use` 命令 `--base` 模式下不再强制要求 profile-id
- 修复 inquirer 导入和空默认值问题
- serve 启动后父进程自动退出
