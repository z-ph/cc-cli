# v2.0.4

add/edit 交互流程新增 CLAUDE_CODE_SUBAGENT_MODEL 提示。

## 新增

- `cc add` / `cc edit` 交互流程新增 `CLAUDE_CODE_SUBAGENT_MODEL` 配置步骤
  - 输入 ANTHROPIC_MODEL 后，自动询问是否与子代理模型一致
  - 支持手动输入自定义子代理模型名称
  - 编辑已有 profile 时自动回填当前值
- `CLAUDE_CODE_SUBAGENT_MODEL` 加入核心环境变量列表，不再出现在扩展变量选择器中
- `.ENV` 加入 `.gitignore`

## 修复

- edit 命令编辑环境变量时正确保留已有的 CLAUDE_CODE_SUBAGENT_MODEL 值
