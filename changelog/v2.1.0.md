# v2.1.0

CLI 命令从 `cc` 重命名为 `zcc`，避免与系统其他 `cc` 命令冲突。新增 `gh:release` 脚本。

## 破坏性变更

- **命令名从 `cc` 改为 `zcc`**：bin 入口、帮助信息、交互提示、文档全部更新。卸载旧版本后重新安装即可
  ```bash
  npm uninstall -g @zphhpzzph/cc-cli
  npm i -g @zphhpzzph/cc-cli@latest
  # 现在使用 zcc 命令
  zcc add myapi -g && zcc myapi
  ```

## 新增

- `pnpm gh:release` 脚本：从 `changelog/vX.Y.Z.md` 读取 release notes，自动创建 GitHub Release
- `scripts/publish-github.js`：支持指定版本号或使用当前 package.json 版本
