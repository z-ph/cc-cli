# v1.0.0

首次发布 — cc-cli 基础功能完整实现。

## 新增

- 完整的 cc-cli 命令行工具实现（`cc <id>`、`list`、`add`、`edit`、`remove`）
- 环境变量选择器，支持分类浏览和有效值枚举
- `ANTHROPIC_BASE_URL` 和 `ANTHROPIC_AUTH_TOKEN` 作为顶级提示字段
- 环境变量注册表外部化为 YAML 配置文件
- 基于 Claude Code settings schema 的配置设计
- `-g/--global` 全局标志支持（list、add、edit、remove 命令）
- `cc edit` 支持重命名配置 ID
- `cc add` 支持留空 model 使用 Claude Code 默认模型
- 一次性备份 + restore 恢复原始配置
- envs/configs 分离架构，新增 config 子命令
- Jest 全面测试套件

## 修复

- 修正 `apikey` 映射到 `ANTHROPIC_AUTH_TOKEN`（非 `ANTHROPIC_API_KEY`）
- 修复 inquirer 导入和 `-t` 选项在子命令上的传递
- 修复空提示默认值问题
